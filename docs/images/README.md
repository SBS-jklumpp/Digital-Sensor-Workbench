## Screenshot Guidelines

1. **Resolution:** Capture at 1920x1080 or higher
2. **Theme:** Use dark mode for consistency
3. **Content:** Show realistic data, not empty states
4. **Format:** PNG preferred, optimize file size
5. **Naming:** Use lowercase with hyphens
